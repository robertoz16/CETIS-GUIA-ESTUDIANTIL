# CETIS-GUIA-ESTUDIANTIL

Un software de escritorio diseñado en *Java* para facilitar la navegación y consulta de información dentro del plantel. Este proyecto utiliza una interfaz gráfica (GUI) para ofrecer una navegación intuitiva entre los edificios, profesores y especialidades.

#  Funcionalidades Principales
1. Navegación Gráfica: Botones interactivos que permiten transiciones fluidas entre secciones.
2. Directorio de Profesores: Organización lógica por edificios para una localización rápida.
3. Mapa de Especialidades: Sección dedicada a la oferta educativa y ubicación de áreas técnicas.
4. Recorrido Visual: Integración de imágenes del plantel para una mejor orientación.

# Tecnologías y Herramientas
1. Lenguaje: Java
2. IDE: Eclipse
3. Interfaz: Swing / WindowBuilder (Modo Gráfico)
4. Paradigma: Programación Orientada a Objetos (POO)

##  Instalación y Ejecución
Para ejecutar este proyecto en tu computadora:
1. Clona este repositorio o descarga el archivo .jar.
2. Asegúrate de tener instalado el *JRE (Java Runtime Environment)*.
3. Ejecuta el archivo principal desde Eclipse o mediante la terminal.
